hexo+github搭建个人博客步骤

工具下载

nodejs下载

node.js官网下载，并配置环境变量
版本信息查看node -v、npm -v

Git下载

下载之前，先在GitHub官网注册GitHub账号（友情提示：需要翻墙）
我的账号：GTY-gty
Git官网下载，并配置环境变量。（友情提示：又要翻墙）
版本信息查看git --version

如果要用hexo等框架制作github pages的话，就要下载git，才能使用hexo的命令
如果不用框架，只要下载github客户端，即git的桌面版
操作只需点点点，非常方便~可以在github上发表博文，发布github pages

要想了解什么是Git，请点击这里

hexo安装

命令行工具：
可以用git自带的git bush（任意位置鼠标右击），或cmd（shift+鼠标右击）
用git bush不用配置git环境变量，用cmd就要配置
用npm下载hexo命令：
npm install hexo-cli -g
查看版本：hexo -v

本地生成hexo文件夹

在合适的文件夹下新建并初始化文件夹：
hexo init "文件夹名"（容易失败，可以试试npm cache clean --force清除缓存）
cd 到新建的文件夹下
安装必备组件：npm install
可以在_config.yml中修改一些配置信息

本地调试命令：hexo s，本地访问地址：localhost:4000

发布到github

安装hexo-deployer-git命令：
npm install hexo-deployer-git --save
修改配置文件：

发布到github pages命令，即部署到git服务器：
hexo clean清除缓存
hexo deploy发布

hexo主题下载：

NexT官网下载，并修改配置文件，详见NexT官网
新建博文命令：hexo new "博文名"
会在source/_posts下生成.md文件，直接编辑.md文件就可以了

参考资料

hexo官网：https://hexo.io
若hexo安装失败，参考资料：https://www.jianshu.com/p/6eba158ffeba
（SSH之类的似乎并不需要）
