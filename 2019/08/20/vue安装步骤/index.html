vue安装步骤

发表于 2019-08-20

记录一下我一个月前vue的安装步骤~

当初的我还对这些乱七八糟的东西一窍不通，当然…现在也是。

下载nodejs

从官网下载nodejs,一路next安装
配置环境变量：path添加nodejs安装路径
node -v和npm -v查看版本

安装webpack

命令行全局安装webpack
npm install webpack -g

安装vue脚手架

版本2：
命令行全局安装vue cli2
npm install -g vue-cli
配置环境变量，安装完后cmd会显示安装路径
vue -V或vue --version查看版本
（若要安装版本3：npm install -g @vue/cli）
至此，vue的安装已经结束。

创建vue新项目

选择一个文件夹，在当前文件夹下打开cmd
初始化项目命令：
vue init webpack "项目名"
（vue cli3命令：vue create "项目名"，vue cli3是否不需要安装webpack？）
一路回车，有Y/n的选Y

安装UI框架

安装element-ui，每个项目安装一次，此框架是基于vue cli2的版本。
文件路径下npm install element -ui -S
VS Code打开package.json文件可以查看vue cli和element ui版本信息

运行

js滚动播报
